
# Spotify Web Player Clone (Frontend Project)

A fully responsive Spotify-inspired web music player built using HTML, CSS, and JavaScript. This project focuses on front-end fundamentals, DOM manipulation, media handling, and responsive UI design without using any external frameworks or libraries.

## Over View

This project simulates the core experience of Spotifyâ€™s web player. Users can browse artists, view song lists, play audio files, control playback, adjust volume, and navigate the UI smoothly across devices.

The application plays locally stored audio files organized by artist folders and dynamically updates the UI based on user interaction
## Technologies

**HTML:** Structure and Layout

**CSS3:** Styling, animations and responsiveness

**JavaScript (ES6):** Logics, events, audio control

**HTML Audio API:** Music Playback

No frameworks, libraries, or backend services are used.

This is a **frontend-only** demo it does **not stream music online**, you provide your own audio files


## Features

### Music Playback

- Play, pause, next, and previous controls

- Automatically plays the next song when the current one ends

- Displays song name and artist

### Dynamic Library

- Click on an artist card to load their songs

- Songs are generated dynamically using JavaScript

- Scrollable song list for better UX

### Progress Bar

- Real-time song progress tracking

- Click or drag to seek within a song

- Displays current time and total duration

### Volume Control

- Adjustable volume slider

- Mute / unmute toggle

- Remembers last volume before muting

- Dynamic volume icon switching

### Responsive Design

- Fully responsive across desktop, tablet, and mobile

- Burger menu for sidebar on small screens

- Smooth animations and transitions
## Run Project

### Songs Assets Download Required

Due to file size and copyright considerations, audio files are **not included** in this repository.

You can download the required song folders from the link below:

https://drive.google.com/file/d/1ujwcqgw6NcjQgmTlkp5y2H0vohfuyEIf/view?usp=sharing

### After Downloading

Extract the Download folder

Clone the repository

Place Songs folder inside Assets/

Open index1.htm in the modern browser

Note: For best results, run using a local server (Live Server in VS Code) to avoid browser autoplay restrictions
